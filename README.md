# cpp-common-utils

## Introduction
This repository contains multiple minimal utility libraries. These libraries are standalone and don't require external dependencies.

Each library is located in a separate folder. In each library folder, you will find all the header/source files required for the library, as well as a `main.cpp` file showing how to use the utility library. 

For example, in the `Logger` folder, you can find the source file `Logger.cpp` and header file `Logger/Logger.h`. There is also the `main.cpp` file containing code examples.

## Usage

You can just copy the code files to your project since there are no external dependencies. Depending on your development environment, you might have to tweak your build configuration.
